## Quick Customization

### Change Your Brand Name
1. Open each HTML file in text editor (Notepad, TextEdit, VS Code)
2. Find and replace "AXIOM" with your brand name
3. Save files

### Change Colors
1. Open `styles.css`
2. Find this section at the top:
```css
:root {
    --color-accent: #C9A962;  /* Change this! */
}
```
3. Replace `#C9A962` with your color (use color picker online)
4. Save file

### Add Your Products
1. Open `products-data.js`
2. Copy an existing product block
3. Change: id, name, category, price, description
4. Save file
5. Refresh browser - new product appears!

## Common Issues & Fixes

### "Products don't show"
**Fix**: Make sure all .js files are in same folder as .html files

### "Cart doesn't save"
**Fix**: Enable cookies/localStorage in browser settings

### "Styles look broken"
**Fix**: Check styles.css is in same folder, clear browser cache

### "Mobile menu stuck open"
**Fix**: Click outside menu or press ESC key

### "Can't edit files"
**Fix**: Use text editor like VS Code (free), not Microsoft Word
